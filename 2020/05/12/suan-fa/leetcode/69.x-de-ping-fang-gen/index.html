    题目简介

    • 题目来源： LeetCode
    • 题目描述：
      实现 int sqrt(int x) 函数。

    计算并返回 x 的平方根，其中 x 是非负整数。

    由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去

    解题思路

    • 由于x的平方跟的整数部分ans是满足$k^2$<=x的，所以，我们可以对K进行二分查找来得到答案；
    • 二分查找的下限定为0，上限定为x，在查找的过程中，比较mid*mid与x的大小关系，根据结果来调整上下限范围。
