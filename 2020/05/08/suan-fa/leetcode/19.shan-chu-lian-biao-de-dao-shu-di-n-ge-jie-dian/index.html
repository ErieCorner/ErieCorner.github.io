    题目简介

    • 题目来源： LeetCode
    • 题目描述：
      给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

    说明：
    给定的 n 保证是有效的。

    • 示例

    解题思路

    • 使用快慢指针slow和fast，当fast走到第K个节点时，slow也开始走，当fast走到链表尾时，slow就走到了倒数第K+1个节点，这时删除slow的下一个节点。
