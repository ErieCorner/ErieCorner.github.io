    题目简介

    • 题目来源： LeetCode
    • 题目描述： 给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

    说明:
    必须在原数组上操作，不能拷贝额外的数组。
    尽量减少操作次数。

    • 示例

    解题思路

    • 双指针法，用记录不是0的元素下标，i遍历数组，当nums[i]!=0的时候，将nums[i]的值给nums[j];并把0 给nums[i];
